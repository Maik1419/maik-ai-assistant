# 🧠 Maik AI Assistant

Assistant domestique intelligent basé sur FastAPI, MQTT et OpenAI.

## 🚀 Fonctionnalités

- API REST avec FastAPI
- Communication MQTT pour capteurs domestiques
- Intégration OpenAI pour intelligence conversationnelle
- Gestion de température via capteurs simulés

## 🛠 Technologies utilisées

- Python 3
- FastAPI
- MQTT (paho-mqtt)
- OpenAI API
- Uvicorn

## ▶️ Lancer le projet

```bash
uvicorn assistant_server:app --reload
